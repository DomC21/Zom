# Zom AI - Stock Analysis Platform

A modern stock analysis platform that combines real-time market data with AI-powered insights.

## Features

- Real-time stock data and technical analysis
- AI-powered market insights and predictions
- Technical indicators (RSI, MACD, Bollinger Bands)
- Fundamental analysis metrics
- Modern, responsive UI with dark theme

## Repository Structure

- `/frontend` - React-based web interface
- `/backend` - FastAPI-based REST API service

## Technologies

- Frontend: React, TypeScript, TailwindCSS
- Backend: Python, FastAPI, WebSocket
- APIs: Polygon.io, Alpha Vantage, OpenAI
